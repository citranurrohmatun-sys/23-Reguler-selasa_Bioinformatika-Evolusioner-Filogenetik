# Detection and Phylogenetic Analysis of Gammacoronavirus

## Background
This project investigates the detection and phylogenetic characterization of Gammacoronavirus from bat samples using Nested PCR targeting the RdRp gene and Oxford Nanopore Technology (ONT).

## Objectives
- Evaluate the effectiveness of ONT Rapid Barcoding Gen-3
- Analyze phylogenetic relationships among Gammacoronavirus sequences
- Support zoonotic surveillance through genomic approaches

## Methods
### Sample
Bat sample (NS 1572)

### Sequencing
- Oxford Nanopore Technology (ONT)
- Rapid Barcoding Kit
- MinION Sequencer

### Bioinformatics Analysis
- FASTQ upload to CZ ID
- Metagenomic pipeline (mNGS)
- MultiBLAST analysis
- Phylogenetic tree construction using MEGA12 Neighbor-Joining

## Results
Phylogenetic analysis showed that Mexico Gammacoronavirus isolates clustered in one clade with high bootstrap values (99–100%), indicating close evolutionary relationships.
